SDIF Sound Description Interchange Format

This is the main page describing the SDIF standard file format. The Sound Description Interchange Format (SDIF) is an established standard for the well-defined and extensible interchange of a variety of sound descriptions including spectral, sinusoidal, time-domain, and higher-level models. SDIF consists of a basic data format framework and an extensible set of standard sound descriptions. The SDIF standard has been created in collaboration by Ircam, CNMAT, and IUA-UPF.
