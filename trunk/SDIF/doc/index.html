Mailing List

The SDIF mailing list allows discussions and provides information about new releases.
  • Use the sympa web interface, or:

  • To subscribe, send a mail to listserv@ircam.fr containing 
    subscribe sdif <first name> <last name>
  • To unsubscribe, send a mail to listserv@ircam.fr containing 
    signoff sdif [e-mail address if different from your reply-address]
  • Send messages to go to the list to sdif@ircam.fr.
