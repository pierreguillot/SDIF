Spectral Envelopes

Matrix IENV SpectralEnvelopeInfo

Description:
Spectral envelopes information matrix, defines interpretation of following 1ENV matrix
Rows: number=1
Columns:
  1. HighestBinFrequency [Hz] frequency of the highest bin of the 1ENV matrix
  2. ScaleType 0 for linear, 1 for logarithmic scaling
  3. BreakFrequency [Hz] break frequency when logarithmic scaling

Matrix 1ENV SpectralEnvelope

Description:
Spectral envelope in sampled representation.
Description (language = fr):
Abradakadabra abrakabum.
Rows:
A row corresponds to one bin of the spectral envelope
Columns:
  1. Env envelope bin
